Oracle 模式支持的数据类型 
=====================================

Oracle 模式所支持的数据类型如下面的列表所示。

* 字符类型

  * VARCHAR2

    
  
  * NVARCHAR2

    
  
  * CHAR

    
  
  * NCHAR

    
  

  

* 数值类型

  * NUMBER

    
  
  * NUMBER_FLOAT

    
  
  * BINARY_FLOAT

    
  
  * BINARY_DOUBLE

    
  

  

* 复杂数据类型

  * Ref Cursor

    
  
  * Struct

    
  
  * Array

    
  
  * RAW

    
  
  * RowId

    
  

  

* 时间日期类型

  * DATE

    
  
  * TIMESTAMP

    
  
  * TIMESTAMP WITH TIME ZONE

    
  
  * TIMESTAMP WITH LOCAL TIME ZONE

    
  
  * INTERVAL

    
  
  * INTERVALDS

    
  
  * INTERVALYM
